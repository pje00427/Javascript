변수와 자료형

* 변수 선언

        <script>;

            변수명;        // 전역변수 (global)

            var 변수명;    // 전역변수 (global)

            function 함수명(){
                var 변수명;     // 지역변수 (local)

                변수명;         // 전역 변수 (global)
            }
        </script>;

전역변수 명과 지역변수명이 동일할 경우

특정 함수 내부에서 동일한 변수명을 호출했을 때
지역변수가 우선권을 가짐
전역변수를 사용하고자 한다면 명시적으로 window또는 this를 붙여 호출

* 자료형

자바스크립트는 변수 선언시 자료형을 별도로 지정하지 않음(자료형의 개념이 없는건 아님)
--> 변수에 대입되는 값(리터럴)에 따라서 알아서 자료형이 자동으로 결정됨

자바스크립트의 자료형

  • Stirng(문자열)
  • number(숫자)
  • boolean(논리값)
  • object(객체)
  • function(함수)
  • undifined(초기화가 안된 변수)
