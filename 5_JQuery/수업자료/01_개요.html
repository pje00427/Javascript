jQuery 개요

* jQuery란 ?

기존에 자바스크립트 언어만으로 구현하기에 복잡했던 구문 작성들을
간소화하기 위해 "존 레식"에 의해 개발된 JavaScript기반 라이브러리(유용한 기능들의 모음집)
즉, JavaScript 언어로 구성된 유용한 함수나 내용들이 이미 만들어져있는 (정의되어있는) 모음집
복잡한 자바스크립트 구문을 짧고 쉽게 개발 가능 (적은 양의 코드로 빠르고, 풍부한 기능 제공)

>> jQuery의 장점
1) DOM과 관련된 스크립트를 쉽게 구현할 수 있다.
2) ajax(비동기) 통신, 이벤트 처리 등을 폭넓게 지원함
3) jQuery와 관련된 확장형 플러그인, 오픈 API등을 다양하게 지원
이를 통해 차트, 슬라이드, 애니메이션을 간단하게 구현할 수 있다.

* jQuery 라이브러리 연결 방법

오프라인 / 온라인 방식이 있음

- jQuery 공식사이트 : 공식사이트

1. jQuery 라이브러리 다운로드를 통한 연결(오프라인)
- .js 파일을 내가 직접 다운받아 저장한 후 해당 저장된 파일 경로 작성하는 방법

2. CDN(Content Delivery Network) 를 통한 연결 (온라인)
- 라이브러리를 제공하는 사이트 주소를 이용하여 파일 경로 지정

* jQuery 시작하기

jQuery 명령은 문서에 dom요소가 다 로드된 후에 실행되어야됨

> 자바스크립트에서의 해당 내용은 

        window.onload = function(){
            실행내용;
            // 해당 문서의 dom요소들 로드되고, 뿐만아니라 이 문서에 연동되어있는
            // 외부문서들 까지 다 가지고오고나서 그리고 실행됨!!
        };
단, javascript에서의 window.onload 구문은 해당 문서에서 단 한번만 작성가능

> jQuery에서의 해당 내용은 3가지로 기술할 수 있음 
        1)
        jQuery(document).ready(function(){
            실행내용;
            // 해당 문서의 dom요소들만 다 로드되면 곧바로 실행됨
        });

        2)
        $(document).ready(function(){ // 보통 제일 많이 쓰임
            실행내용;
        });

        3)
        $(function(){ // 제일 축약형
            실행내용;
        });
jQuery에서의 해당 구문들은 해당 문서에서 여러번 작성가능
